The org.janux.bus.security package, soon to be renamed the org.janux.authorization package, contains interfaces that define a very flexible Permission-Based scheme for doing Authorization within an application; this scheme makes it possible to define whatever permissions suit a specific domain, and aggregate them into existing or new roles without having to make code changes; the key interfaces of this package are {@link org.janux.bus.security.PermissionsCapable}, {@link org.janux.bus.security.Account}, {@link org.janux.bus.security.Role}, and {@link org.janux.bus.security.PermissionContext}.
