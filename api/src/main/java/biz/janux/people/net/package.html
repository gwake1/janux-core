The biz.janux.people.net package that contains classes representing the network-based forms of {@link biz.janux.people.ContactMethod ContactMethods}, namely {@link biz.janux.people.net.Uri Uniform Resource Identifiers (URI)}, and its subclasses {@link biz.janux.people.net.Email} and {@link biz.janux.people.net.Url Uniform Resource Locators (URL)}.
